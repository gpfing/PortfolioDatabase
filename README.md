To download:

1. Go to - https://github.com/gpfing/PortfolioDatabase
2. Create a fork of the project
3. Copy the SSH code and input it into your machine
4. Open the console and navigate to the project file
5. Type 'npm run dev' into the command line
6. Open the host link within the console
7. Enjoy!

You can add your own project to the list, and then sort the projects by name or time for completion. It even lets you only view projects you've completed. If the list gets too long, please use the search bar to find your projects quicker!
